# MCP Servers and Devcontainer

This repository contains multiple projects used to run and test MCP-related development workflows.

## Projects

- **docker-mcp-gateway**: Docker-based MCP Gateway setup that routes tools/skills to multiple MCP servers and centralizes local integration.
- **semut-codebase-index-2**: Codebase indexing and MCP client/server project for searching, analyzing, and interacting with indexed source code.
